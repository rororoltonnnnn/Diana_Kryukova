# Додыр (Мобильное приложение)
Треск сверчков...

Пользователи:
1)Бугалтер 
2)Директор 
3)Касса   
  Блок лист. VIP клиенты. Акции. Тарифы. Статистика и прогноз. В 1С все заказы. Данные о клиенте и заказов. 
  
4)Клиент 
  Авторизация: номер телефона. Вкалдка с оплченными/неоплаченными услугами с чеком. Оплата картой, наличкой, СПБ. Отмена услуги в любое время, но не во время оказания услуги. Акции. Тарифы. Без самообслуживания. История услуг с возможностью повторить услугу. Оповещения. Статус клиента. Обратная связь. Вкладка (карта) с точками. Указывать размер машыны. Редактрирование заказа.


Корсаков мойка на 5 боксов, Долинск 7, Южно-Сахалиснк 10 боксов, машины уходят от 10 до 15 минут.

Состивим диаграмму USE CASE, диагрумму последовательностей C4
